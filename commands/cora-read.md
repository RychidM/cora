---
description: Read the full content of a vault file or entry.
skill: cora-read
---

Use the `cora-read` skill to return the complete content of one resolved
target — a project file, an idea, a single issue, a brand file, or a log
entry — rather than search snippets.

Argument: what to read, e.g.
- `/cora-read agentwatch overview`
- `/cora-read product ideas`
- `/cora-read ISSUE-003`
- `/cora-read brand profile`

If the target is ambiguous, the skill lists candidates and asks.
Read-only — never writes.
